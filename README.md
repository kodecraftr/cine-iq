# 🎬 Movie Recommender Engine

A hybrid, explainable movie recommendation system combining collaborative filtering,
content-based filtering, sentiment re-ranking, and natural-language explanations.

---

## Setup

```bash
# 1. Clone and install
pip install -r requirements.txt

# 2. Run all phases in order
python src/data/preprocess.py                          # Phase 1
python src/models/collaborative.py --sample-frac 0.1  # Phase 2 (quick test)
python src/models/content_based.py                     # Phase 3
python src/models/ensemble.py --user-id 42             # Phase 4
python src/models/sentiment.py                         # Phase 5
python src/explainability/explainer.py --user-id 42   # Phase 6
```

## Architecture

```
User Request
     │
     ▼
FastAPI Gateway  (/recommend  /similar)
     │
     ├── Collaborative Filter (SVD via Surprise)
     │       └── top-200 candidate generation
     │
     ├── Content-Based Filter (TF-IDF + cosine similarity)
     │       └── content scores for candidates
     │
     ├── Weighted Ensemble  (α·collab + β·content)
     │       └── hybrid ranking
     │
     ├── Sentiment Re-Ranker (VADER / DistilBERT)
     │       └── nudge by audience reception signal
     │
     └── Explainability Layer (rule templates + LIME)
             └── human-readable reason per recommendation
```

## Key Hyperparameters

| Parameter | Default | Meaning |
|---|---|---|
| `alpha` | 0.6 | Collaborative filter weight in ensemble |
| `beta` | 0.4 | Content-based weight (= 1 - alpha) |
| `gamma` | 0.15 | Sentiment nudge strength |
| `n_factors` | 100 | SVD latent factors |
| `n_epochs` | 30 | SVD training epochs |

Tune `alpha` via `ensemble.py --tune` and `n_factors/n_epochs` via `collaborative.py --tune`.
All runs are tracked in MLflow.
